== Description ==

This plug-in you can change words font attributes easily. You only need to write correct CSS codes for font colors, font faces and font sizes.

The Color Keywords SEO plugin was developed so that important blog post keywords on a WordPress blog could be optimized using the HTML <strong>, <em> and <u> tags.

Related Links:

* <a href="http://seoarsivi.com/" title="SEO">Plugin Homepage</a>
* <a href="http://seoarsivi.com/plugins/color-keywords.seo" title="SEO">Plugin Page</a>

Colors
You can write color codes in two ways.

1. Color name : blue, red 
1. Color hexadecimal (HEX) code : #0000CD,#00FF00

this plug-in also supports other CSS color codings.

Font section, you can manage your posts font names. You only need to write the font name.
Examples

1. "Times New Roman", Times, serif
1. Arial

Size section you can manage your fonts size.
Examples

1. 250%
1. 12px


== Installation ==

1. Upload `color-keywords` folder to the `/wp-content/plugins/` directory
1. Activate the plug-in through the `Plug-ins` menu in WordPress

1. Visit your Color Keywords Settings (*Settings/Color Keywords*)
1. Configure options
1. That is it!
